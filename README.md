# Micro Front End App example

App using [module federation](https://webpack.js.org/concepts/module-federation/)

## Installation


Admin app

```bash
cd admin
npm install
npm start
```

Order app

```bash
cd order
npm install
npm start
```


## Usage

```bash
cd admin
npm start
```

```bash
cd order
npm start
```
